# AuctionHub Ultra

- Express + MongoDB + Socket.IO
- JWT auth (cookies)
- Sell Item with JPEG/PNG upload (multer) + optional URL
- Explore, Item detail, My Bids
- Tailwind polished UI

## Setup
```bash
cp .env.example .env
npm install
npm run seed   # adds 50 demo items
npm run dev
# open http://localhost:5000
```
